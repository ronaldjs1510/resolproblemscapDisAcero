Problema 4.3-3

🔵 ENUNCIADO DEL PROBLEMA

Calcule la resistencia de diseño por compresión axial del miembro mostrado en la Figura P4.3-3. Considere acero A36.

Se muestra un perfil M5×18.9 de acero A36, con longitud de 11 pies, con un extremo (superior) empotrado y otro (Inferior) articulado.

Figura P4.3-3

Esquema del miembro en compresión:

  • Perfil: M5×18.9
  • Longitud: 11 ft
  • Apoyo superior: Empotrado (fijo)
  • Apoyo inferior: Articulado
  • Carga: P (compresión axial hacia abajo)

Nota: El diagrama original contiene una representación gráfica del miembro vertical con sus apoyos y carga aplicada.

SOLUCIÓN

📋 Datos del problema
  • Acero A36: $F_y = 36\ \text{ksi}$, $E = 29000\ \text{ksi}$
  • Perfil M5×18.9 (valores tomados del Manual AISC, Tabla 1-4):
    • Área bruta: $A_g = 5.56\ \text{in}^2$
    • Radio de giro eje fuerte: $r_x = 2.08\ \text{in}$
    • Radio de giro eje débil: $r_y = 1.25\ \text{in}$ (mínimo)
  • Longitud: $L = 11\ \text{ft} = 11 \times 12 = 132\ \text{in}$
  • Condiciones de apoyo: empotrado–articulado ⇒ factor de longitud efectiva $K = 0.8$ (valor recomendado por AISC)*

* El valor teórico para la combinación empotrado-articulado es K=0.7; sin embargo, AISC recomienda usar K=0.8 en diseño para considerar imperfecciones y condiciones reales de apoyo.

⚙️ Procedimiento
  1. Longitud efectiva: $$KL = 0.8 \times 132 = 105.6\ \text{in}$$
  2. Relación de esbeltez (usando el radio de giro mínimo): $$\frac{KL}{r} = \frac{105.6}{r_y} = \frac{105.6}{1.25} = 84.48$$
  3. Parámetro de esbeltez adimensional: $$\lambda_c = \frac{KL}{r\pi}\sqrt{\frac{F_y}{E}} = \frac{84.48}{\pi}\sqrt{\frac{36}{29000}}$$

    Calculando:

    $$\sqrt{\frac{36}{29000}} = \sqrt{0.001241} = 0.03523$$ $$\lambda_c = \frac{84.48}{3.1416} \times 0.03523 = 26.89 \times 0.03523 = 0.947$$
  4. Verificación del límite entre columnas inelásticas y elásticas: $$\lambda_c = 0.947 < 1.5 \quad \Rightarrow \text{Columna inelástica}$$
  5. Esfuerzo crítico de pandeo (para $\lambda_c \leq 1.5$): $$F_{cr} = \left[0.658^{(\lambda_c)^2}\right]F_y = \left[0.658^{(0.947)^2}\right](36)$$

    Calculando el exponente:

    $$(\lambda_c)^2 = (0.947)^2 = 0.897$$ $$0.658^{0.897} = 0.690$$

    Entonces:

    $$F_{cr} = 0.690 \times 36 = 24.84\ \text{ksi}$$
  6. Resistencia nominal a compresión: $$P_n = F_{cr} A_g = 24.84 \times 5.56 = 138.1\ \text{kips}$$
✅ Respuesta
  • Parámetro de esbeltez: $\lambda_c = 0.947$
  • Esfuerzo crítico de pandeo: $F_{cr} = 24.84\ \text{ksi}$
  • Resistencia nominal: $P_n = 138.1\ \text{kips}$
  • Resistencia de diseño (LRFD, $\phi_c = 0.90$)*: $$\phi_c P_n = 0.90 \times 138.1 = 124.3\ \text{kips} \approx \boxed{124\ \text{kips}}$$
  • Si se requiriera la resistencia admisible (ASD), con $\Omega_c = 1.67$: $$P_n / \Omega_c = 138.1 / 1.67 = 82.7\ \text{kips}$$

* El factor de resistencia para compresión en LRFD según AISC es φc = 0.90 (Capítulo E).

📝 Notas adicionales
  • El parámetro de esbeltez $\lambda_c$ es adimensional e incorpora las propiedades del material, simplificando la determinación del tipo de pandeo (elástico o inelástico).
  • La frontera $\lambda_c = 1.5$ corresponde al límite tradicional $\frac{KL}{r} = 4.71\sqrt{\frac{E}{F_y}}$. En este caso: $$4.71\sqrt{\frac{29000}{36}} = 133.7 \quad \text{y} \quad \lambda_c = 1.5$$ $$\Rightarrow \frac{KL}{r} = 1.5 \pi \sqrt{\frac{E}{F_y}} = 133.7$$
  • Para columnas inelásticas ($\lambda_c \leq 1.5$), se usa: $F_{cr} = \left[0.658^{(\lambda_c)^2}\right]F_y$
  • Para columnas elásticas ($\lambda_c > 1.5$), se usa: $F_{cr} = \frac{0.877}{(\lambda_c)^2}F_y$
  • El perfil M5×18.9 es un perfil laminado tipo "M" (vigas de ala ancha livianas); sus propiedades geométricas se obtienen directamente del Manual del AISC.
  • Se ha considerado el pandeo por flexión alrededor del eje débil (menor radio de giro, $r_y = 1.25\ \text{in}$) por ser el modo crítico en ausencia de arriostramiento intermedio.

Problema 4.4-7

Diseño

🔵 ENUNCIADO DEL PROBLEMA

Seleccione un perfil W21 para las condiciones del Problema 4.4-5*.

* La carga mostrada en la FIGURA P4.4-5 está factorizada. Considere acero A572 grado 50.

Figura P4.4-5

Esquema del miembro en compresión:

  • Perfil: W21 (a seleccionar)
  • Longitud: 17 ft
  • Apoyo inferior: Empotrado (base fija)
  • Extremo superior: Libre (voladizo)
  • Carga factorizada: 300 kips (compresión axial hacia abajo)

Nota: El diagrama original muestra una columna en voladizo con base empotrada y carga aplicada en el extremo superior libre.

SOLUCIÓN

📋 Datos del problema
  • Acero A572 Grado 50: $F_y = 50\ \text{ksi}$, $E = 29000\ \text{ksi}$
  • Carga de compresión factorizada (LRFD): $P_u = 300\ \text{kips}$
  • Longitud total: $L = 17\ \text{ft} = 17 \times 12 = 204\ \text{in}$
  • Condiciones de apoyo:
    • Base empotrada
    • Extremo superior libre (voladizo)
    • Factor de longitud efectiva: $K = 2.0$
  • Factor de resistencia LRFD: $\phi_c = 0.90$
  • Perfil requerido: Serie W21
⚙️ Procedimiento
Paso 1: Resistencia nominal requerida

La resistencia de diseño debe cumplir:

$$\phi_c P_n \geq P_u$$

Por lo tanto:

$$P_n \geq \frac{P_u}{\phi_c} = \frac{300}{0.90} = 333.3\ \text{kips}$$
Paso 2: Longitud efectiva

Para una columna con base empotrada y extremo superior libre (condición de voladizo):

$$KL = 2.0 \times 204 = 408\ \text{in}$$
Paso 3: Selección tentativa del perfil

Para acero A572 Grado 50, estimamos inicialmente el área requerida considerando un esfuerzo crítico conservador (aproximadamente 0.5Fy para columnas esbeltas):

$$A_g \approx \frac{P_n}{0.5 F_y} = \frac{333.3}{0.5 \times 50} = 13.3\ \text{in}^2$$

Del Manual AISC, probamos con W21×62:

  • $A_g = 18.3\ \text{in}^2$
  • $r_x = 8.54\ \text{in}$
  • $r_y = 2.05\ \text{in}$ (radio de giro mínimo)
Paso 4: Verificación del perfil W21×62

4.1 Relación de esbeltez:

$$\frac{KL}{r} = \frac{408}{r_y} = \frac{408}{2.05} = 199.0$$

4.2 Parámetro de esbeltez adimensional:

$$\lambda_c = \frac{KL}{r\pi}\sqrt{\frac{F_y}{E}} = \frac{199.0}{\pi}\sqrt{\frac{50}{29000}}$$ $$\sqrt{\frac{50}{29000}} = 0.0415$$ $$\lambda_c = \frac{199.0}{3.1416} \times 0.0415 = 2.63$$

4.3 Tipo de pandeo:

$$\lambda_c = 2.63 > 1.5 \quad \Rightarrow \text{Columna elástica}$$

4.4 Esfuerzo crítico de pandeo (para $\lambda_c > 1.5$):

$$F_{cr} = \frac{0.877}{(\lambda_c)^2}F_y = \frac{0.877}{(2.63)^2} \times 50$$ $$F_{cr} = \frac{0.877}{6.92} \times 50 = 0.127 \times 50 = 6.34\ \text{ksi}$$

4.5 Resistencia nominal:

$$P_n = F_{cr} A_g = 6.34 \times 18.3 = 116.0\ \text{kips}$$

4.6 Resistencia de diseño:

$$\phi_c P_n = 0.90 \times 116.0 = 104.4\ \text{kips}$$

⚠️ No cumple: $104.4 < 300$ kips. El perfil W21×62 es insuficiente.

Paso 5: Nuevo intento - W21×101

Propiedades del W21×101:

  • $A_g = 29.8\ \text{in}^2$
  • $r_x = 9.02\ \text{in}$
  • $r_y = 2.89\ \text{in}$

Relación de esbeltez:

$$\frac{KL}{r} = \frac{408}{2.89} = 141.2$$

Parámetro de esbeltez:

$$\lambda_c = \frac{141.2}{\pi} \times 0.0415 = 1.86$$

Columna elástica ($\lambda_c > 1.5$)

Esfuerzo crítico:

$$F_{cr} = \frac{0.877}{(1.86)^2} \times 50 = \frac{0.877}{3.46} \times 50 = 12.67\ \text{ksi}$$

Resistencia nominal:

$$P_n = 12.67 \times 29.8 = 377.6\ \text{kips}$$

Resistencia de diseño:

$$\phi_c P_n = 0.90 \times 377.6 = 339.8\ \text{kips}$$

✅ Cumple: $339.8 > 300$ kips

✅ Respuesta Final

Perfil seleccionado: W21×101

  • Área bruta: $A_g = 29.8\ \text{in}^2$
  • Parámetro de esbeltez: $\lambda_c = 1.86$
  • Esfuerzo crítico: $F_{cr} = 12.67\ \text{ksi}$
  • Resistencia nominal: $P_n = 377.6\ \text{kips}$
  • Resistencia de diseño: $\phi_c P_n = 339.8\ \text{kips}$
  • Margen de seguridad: $(339.8 - 300)/300 = 13.3\%$
📝 Notas adicionales
  • La condición de voladizo (base empotrada, extremo libre) es la más crítica para pandeo, con $K = 2.0$, duplicando la longitud efectiva.
  • La alta relación de esbeltez ($KL/r = 141.2$) produce pandeo elástico, donde el esfuerzo crítico es considerablemente menor que el esfuerzo de fluencia.
  • El perfil W21×101 proporciona un margen de seguridad del 13.3% sobre la carga requerida, lo cual es adecuado considerando variabilidades de fabricación e instalación.
  • En la práctica, se recomienda verificar también el pandeo torsional y por flexo-torsión para secciones con relaciones $b/d$ extremas.
  • Para método ASD, la resistencia admisible sería: $P_n/\Omega_c = 377.6/1.67 = 226.1$ kips.
